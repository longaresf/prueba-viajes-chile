# Prueba-Viajes-Chile

## Description
This repository contains a web application for a travel company in Chile. The application is built using HTML and static content.

## Tech Stack
- HTML

## Usage
To view the application:
1. Open `index.html` in your web browser.
2. Navigate to the directory containing this file (`prueba-viajes-chile`) on your local machine or a remote server, then open `index.html`.

---

**Note:** This repository is intended for demonstration purposes and does not include any backend functionality or dynamic content generation.

This README.md document provides clear instructions for users to view the static web application contained within the `prueba-viajes-chile` directory.
